SpringBoot整合ES各种问题解决

2021-04-15
N 人看过
字数：557字 | 预计阅读时长：3分钟

一、No Available Nodes

问题原因：
1、可能是版本不兼容，例如我的 SpringBoot 版本是 2.1.8.RELEASE，那么对应的 ES 版本应该在 6.2.2 到 6.8.12 之间，这里选择了 ES:6.8.4 版本

版本对应表：https://docs.spring.io/spring-data/elasticsearch/docs/4.2.0/reference/html/#preface.requirements
2、在 elasticsearch.yml 中添加上配置

二、使用 ElasticsearchTemplate 提示找不到 Caused by: java.lang.ClassNotFoundException: org.elasticsearch.action.admin.indices.mapping.put.PutMappingResponse

按照知乎老哥的回答，应该将 pom.xml 中的 properties 中的 ES 相关版本去掉，这里我将填写的注释掉了，就好用了


附录-依赖信息：
